# 📊 Google Ads Performance Dashboard

## 🔍 Project Overview
The Google Ads Performance Dashboard provides a comprehensive analysis of digital marketing campaigns by tracking key performance indicators such as cost, impressions, clicks, CTR, and CPC. It enables stakeholders to monitor campaign efficiency, identify trends over time, and optimize ad spend for better marketing outcomes.

---

## 🎯 Objectives
- Analyze overall campaign performance across multiple years (2024–2026)  
- Monitor key advertising metrics like impressions, clicks, and cost  
- Evaluate campaign effectiveness using CTR and CPC  
- Identify trends in cost and engagement over time  
- Compare performance across different campaign types (Search, Display, Video, Demand Gen)  
- Enable data-driven decision-making to improve ROI  

---

## 📈 Key Metrics (KPIs)
- **Total Cost:** $1.6M  
- **Total Impressions:** 37M  
- **Total Clicks:** 2M  
- **Click Through Rate (CTR):** 4.15%  
- **Cost Per Click (CPC):** 1.03  

---

## 📊 Dashboard Features
- Year-wise filter (2024, 2025, 2026) for dynamic analysis  
- KPI cards for quick overview of campaign performance  
- Monthly cost trend to track spending patterns across the year  
- Impressions & CTR over time to evaluate engagement trends  
- Campaign type analysis (Search, Display, Video, Demand Gen)  
- Cost distribution by campaign type for spend comparison  
- CPC analysis by campaign type to assess cost efficiency  
- Interactive visuals with drill-down and filtering capabilities  

---

## ▶️ How to Use
1. Download the `.pbit` file  
2. Open in Power BI Desktop  
3. Connect to your own dataset or update the data source  
4. Refresh to view the dashboard

## 💡 Key Insights
- August shows the highest total cost, indicating a peak in campaign spending  
- Impressions remain relatively stable, but CTR fluctuates, suggesting varying engagement quality  
- April shows a spike in CTR, indicating highly effective campaigns during that period  
- Search campaigns contribute the majority of total cost, dominating overall spend  
- Display and Video campaigns have lower spend but also lower CPC, making them cost-efficient  
- CPC peaks in certain months (notably April), which may indicate competitive bidding or high-value keywords  
- Higher spend does not always result in higher CTR, highlighting optimization opportunities  

---

## 🔒 Data Privacy Note
This repository contains a `.pbit` file (Power BI template), which does **not include any underlying data**. Users are required to connect their own dataset to view the dashboard.
